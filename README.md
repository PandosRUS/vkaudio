## Поиск в общем списке
GET запрос:
```
/API.php?method=search&text=`поисковой_запрос`
```
Ответ
```js
[
    'success',
    [
        {
            artist: "Исполнитель"
            name: "Название"
            img_min: "URL изображения 80х80"
            img: "URL изображения 150х150"
            hash: "Хэш аудио, передается при скачивании"
        },
        {
            ...
        }
    ]
]
```
